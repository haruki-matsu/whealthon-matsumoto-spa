## セットアップ方法

1. リポジトリをクローンします：  
git clone git@github.com:haruki-matsu/whealthon-matsumoto-spa.git
cd whealthon-matsumoto-spa

2. Docker コンテナをビルドして実行します：  
docker-compose up --build


## アプリケーションへのアクセス

アプリケーションはデフォルトでポート 8080 で実行されます。  
ブラウザで以下の URL にアクセスしてください：  
http://localhost:8080
